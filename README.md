# learning-management-RA-public
This is a public version of a project I completed for coursework. Original repository can be demonstrated upon potential employer request. 

This is a learning management website similar to "Canvas" built using C# and Blazor.

**Features**
  - Teacher functions
    - Course management
      - Create, edit, duplicate, and delete courses. Set semester start and stop dates.
      - Courses may contain announcements, roster, assignments, grades, and modules.
      - Assignments have due dates.
      - Course content may contain string, files, or a quiz question.
      - Students can be added to global roster list as well as course roster.
      - Assignment submissions can be graded and grading scale can be changed for a course.
   - Student functions
     - View course content, roster, and student grades.
     - Submit responses to course content.

**Prerequisites**

    .NET 10.0 SDK 
    Visual Studio 2022, Rider, or VS Code

**Installation and Running**
    
    Clone repository
    cd learning_management_RA
    dotnet restore
    dotnet run
    Open http://localhost:xxxx/ in browser (address specified by console output)
